# Sensores y Actuadores

Repositorio de prácticas desarrolladas en Arduino mediante simulación en Wokwi para el aprendizaje de sensores, actuadores y sistemas de automatización.

## Tecnologías

- Arduino
- C/C++
- Wokwi
- Git
- GitHub

## Proyectos

### 1. Sensor Ultrasónico HC-SR04
Medición de distancia utilizando ultrasonido.

### 2. Sensor de Temperatura
Lectura y monitoreo de temperatura.

### 3. Servo Motor
Control angular de posición.

### 4. Motor DC
Control de encendido y apagado de motores de corriente continua.

## Competencias desarrolladas

- Lectura de sensores
- Procesamiento de señales
- Control de actuadores
- Automatización básica
- Documentación técnica
- Programación de sistemas embebidos
